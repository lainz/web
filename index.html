Introducción

La web fue inventada por Tim Berners Lee en la decada del 90.

Actualmente tenemos HTML, CSS y JavaScript para trabajar el desarrollo de un sitio web.

HTML

HTML en inglés Hypertext Markup Language o en español Lenguaje de Marcas de Hipertexto, es el lenguaje que se utiliza para mostrar el contenido de la web. Consiste de etiquetas que indican al navegador que mostrar.

HTML - Estructura básica del archivo

HTML es texto, por lo que se puede crear con el bloc de notas o con Visual Studio Code, entre otros.

La extensión del archivo debe ser ".html".

El archivo HTML tiene el siguiente formato básico:


<!DOCTYPE html>
<html>
<head>
<title>Mi sitio web</title>
</head>
<body>
</body>
</html>

El archivo inicia con la etiqueta !DOCTYPE que indica que se trata de un archivo HTML5.

Luego viene la etiqueta html que indica donde inicia y donde termina el archivo html.

Dentro de html tenemos head y body.

La etiqueta head contiene el título de la página en la etiqueta title.

La etiqueta body contiene el cuerpo de nuestro archivo html, donde iran título, navegación, articulos, imágenes, tablas, etc.
